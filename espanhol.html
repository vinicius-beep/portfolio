Proyectos


USBLOCK

USBlock es un software de seguridad que controla el acceso a dispositivos USB, bloqueando unidades flash no autorizadas, verificando automáticamente las unidades conectadas y evitando la entrada de archivos maliciosos. El proyecto es gratuito, compatible con Windows y se desarrolló en C utilizando la biblioteca GTK4 y CSS para la interfaz gráfica. Su objetivo es ayudar a las pequeñas empresas a mantenerse seguras.


RODEI+

El sistema de Control de Gastos Vehicular ofrece una plataforma completa para gestionar gastos, mantenimientos e historial de vehículos. Permite visualizar indicadores, generar informes detallados, monitorear costos mensuales y anuales y exportar datos en PDF o Excel, ayudando a los usuarios a mantener la organización y el control financiero de forma simple y eficiente. Rodei+ servirá para facilitar la vida de los conductores.
